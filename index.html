The Basic Language of the Web: HTML

Posted by Laura-Jones on Monday, June 21st 2027

All modern websites and applicaions are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML, is all about, and why you too should learn it.

What is HTML

HTML stands for HyperText Markup Laguage. It's a markup language that web developers use to structure ans describe the content of a webpage (not a programmig language).

HTML consist of elements that describe different types of content: paragraphs, links, heading, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML each element is made up of 3 parts:

  1. The Opening tag
  2. The closing tag
  3. The actual element
